# Theme Tools

Utilities to swap colors and apply hex code updates across project files.

## Scripts

### 1. `color_palette_hex.py`
Re-themes color palettes on target React/Tailwind/CSS source files by mapping original classes or standard color values directly into custom aesthetic hex values.

* **Usage**:
  Edit the target `file_path` in the script, then execute:
  ```bash
  python3 color_palette_hex.py
  ```

---

### 2. `color_swap.py`
Converts Tailwind color configurations (e.g. slate classes to stone classes) and ambient glow classes systematically.

* **Usage**:
  Edit the target `file_path` in the script, then execute:
  ```bash
  python3 color_swap.py
  ```
