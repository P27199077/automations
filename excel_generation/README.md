# Excel Generation

Utilities to generate stylized task, training, and schedule planner documents in Microsoft Excel format.

## Scripts

### 1. `create_life_reset_excel.py`
Generates a stylized, color-themed, gamified 60-Day life-reset challenge tracker spreadsheet featuring checklists, dates, and pre-populated daily targets (Hyrox workout plans, LeetCode tasks, side project roadmap focus, and custom tasks).

* **Usage**:
  Edit the destination path in `output_path`, then execute:
  ```bash
  python3 create_life_reset_excel.py
  ```
